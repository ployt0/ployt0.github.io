View webp files

On Ubuntu 20.04.

gthumb

I'd forgotten about this but vaguely recall installing it before where it performed as an image viewer replacement, being able to move left and right between files, with a little more functionality and clutter than I was used to.

It looks light-weight from itsfoss where they describe it as a photo viewer application". 

sudo apt-get install gthumb

I can see it will take 32MB of disk on my relatively clean Ubuntu 20.04.

imagemagick

The popular batch processing tool. It requires 19.4MB of disk from me and webp which in turn needs 631KB.

GIMP

My go-to image editor, already installed. GIMP 2.10 already supports import and export.

Firefox

Brilliant. The default Ubuntu browser of course handles viewing, but only one at a time.

Eye Of Gnome

EoG is the default "Image Viewer" in gnome. This SO answer from June 2021 provides comprehensive instructions for enabling webp support. No need for scary PPA repositories, just one "little known" github repo.

Requires some building, with meson, and ninja.

JPEG XL

I gleaned from the SO answer above, by Pablo Bianchi. Lossless compression of jpeg, so essentially jpeg but better. If I hadn't just solved compatibility issues with the above options, I'd hold out for this.
